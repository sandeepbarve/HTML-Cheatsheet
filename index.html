This HTML Cheatsheet Is For You!
  1. HTML Boilerplate: This is the basic HTML Boilerplate code !

    let's take look at a basic example.

    <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
    
</body>
</html>


  2. The <title> tag is the title for the web page. This text is shown in the browser's title bar. 
     <title>HTML 5 Boilerplate</title>


  3. CSS stylesheet This code will link your custom CSS to the HTML page.

    rel="stylesheet"

    <link rel="stylesheet" href="style.css">


  4. Script tags in HTML
    5. <script src="index.js"</script>


Conclusion


A boilerplate in HTML is a template you will add at the start of your project. This starter code contains important information like the doctype, metadata, external stylesheets and script tags.
